 Dashboard Financeiro – Power BI

 Objetivo
Analisar receitas, despesas e desempenho financeiro da empresa.

 Indicadores desenvolvidos
- Receita total
- Despesas totais
- Resultado líquido
- Evolução mensal
- Comparativo receita vs despesa

 Ferramentas utilizadas
- Power BI
- Modelagem de Dados

 Contexto
Projeto voltado para análise financeira e suporte à tomada de decisão estratégica.
